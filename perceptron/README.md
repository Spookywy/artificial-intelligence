# Perceptron

A simple Perceptron implementation

# What is a Perceptron?

A perceptron is a binary classifier

It consists of:
- a set of inputs
- one weight per input
- a bias
- a weighted sum (weighted sum = (input1 * weight1) + (input2 * weight2) + ... + (inputn * weightn) + bias)
- an activation function (usually a step function) to output either 1 or 0
